private_table¶

PrivateTable

Private Table class which uses pandas dataframe as the backend.
class PrivateTable(dataframe, data_domains, total_privacy_budget, delta_prime=0.5)[source]¶

Bases: object

Private Table class which uses pandas dataframe as the backend.

Differentially private mechanisms for a data table.

Supported statistical functions:

  • mean: the private mean.

Create a private data table.

Parameters

  • dataframe (-) – the data source (expected a pandas dataframe).

  • data_domains (-) – specify the set of all posible value for each data column. It is a map from column_name to a data_domain.

  • total_privacy_budget (-) – the total privacy privacy that can be consumed by the private table. When is there is no privacy budget left, stop answering queries.
