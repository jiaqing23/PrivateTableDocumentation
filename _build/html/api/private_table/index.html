private_table¶

PrivateTable

Private Table class which uses pandas dataframe as the backend.
class PrivateTable(dataframe, data_domains, total_privacy_budget, delta_prime=0.5)[source]¶

Bases: object

Private Table class which uses pandas dataframe as the backend.

Differentially private mechanisms for a data table.

Supported statistical functions:

  • mean

  • std

  • var

  • min

  • max

  • median

  • mode

  • histogram

Parameters

  • dataframe (DataFrame) – The data source

  • data_domains (Dict[str, DataDomain]) – Specify the set of all posible value for each data column. It is a map from column_name to a data_domain.

  • total_privacy_budget (PrivacyBudget) – The total privacy budget that can be consumed by the private table. When is there is no privacy budget left, stop answering queries.
