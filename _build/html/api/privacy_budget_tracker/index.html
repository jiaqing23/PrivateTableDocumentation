privacy_budget_tracker¶

PrivacyBudgetTracker

Base class of privacy budget tracker.

SimplePrivacyBudgetTracker

Privacy budget tracker that use simple composition theorem to update consumed privacy budget.

AdvancedPrivacyBudgetTracker

Privacy budget tracker that use advance composition theorem to update consumed privacy budget.

MomentPrivacyBudgetTracker

Privacy budget tracker that use moment accountant (https://arxiv.org/pdf/1607.00133.pdf) to update consumed privacy budget.

update_privacy_loss(sampling_ratio, sigma, steps, moment_order=32, target_eps=None, target_delta=None)[source]¶

Calculate and update privacy loss. Must specify exactly either one of target_eps or target_delta.

Parameters

  • sampling_ratio (float) – Ratio of data used to total data in one step

  • sigma (float) – Noise scale

  • steps (int) – Number of update performed

  • moment_order (int) – Maximum order of moment to calculate privacy budget, defaults to 32

  • target_eps (Optional[float]) – Target value of \(\epsilon\), defaults to None

  • target_delta (Optional[float]) – Target value of \(\delta\), defaults to None
